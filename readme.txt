Operating system used: Windows 11
IDE used: Visual Studio Code, using the MinGW compiler

Running the executables: After compiling and generating the executables using CMake,
 no additional arguments are required other than the file name, e.g. ./imgDisplay

Project Tree:
project1
	->bin: executables
	->data: images and videos
	->build: CMake files
	->include: filter.hpp and filter.cpp
	->src: imgDisplay.cpp and vidDisplay.cpp

Instructions for extenstions: The wiki report describes the use of the extensions, which are duo-tone filters,
 color pop effect, and video clip saving using the 'w/e/r', 'p', and 'o' keys respectively
